# How to Add Project to LinkedIn Profile

---

## Step-by-Step Instructions

### 1. Go to LinkedIn Profile
- Log into LinkedIn
- Click "Add profile section" → Projects

### 2. Fill in the Fields:

---

**Project Name:**
```
Insurance Claims Fraud Detection System
```

**Description:**
```
Built a production-grade data pipeline for detecting fraudulent insurance claims using machine learning.

Key Features:
• ETL Pipeline - Extract, transform, validate, and load insurance claims data to PostgreSQL
• ML Model - RandomForest classifier with 78% accuracy and 0.785 AUC
• REST API - FastAPI with 8 endpoints, authentication, rate limiting, and caching
• Dashboard - Streamlit analytics with fraud visualizations
• Automation - Prefect scheduling for daily ETL and weekly model retraining
• Monitoring - Prometheus metrics, health checks, and alerting
• DevOps - Docker containerization, GitHub Actions CI/CD, 36 unit tests
• Data Quality - Validation rules, data lineage tracking, backup/restore

Tech Stack: Python, PostgreSQL, FastAPI, scikit-learn, Docker, Prefect, Prometheus

This project demonstrates end-to-end data engineering skills suitable for Data Engineer roles.
```

## Alternative: Add as "Featured"

If you want maximum visibility:

1. Go to your profile
2. Click "Add profile section" → Recommended → Featured
3. Select this project and your best blog/post about it

This puts it at the top of your profile!
